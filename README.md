# Insurance Cost Analysis

This project analyzes an insurance dataset to predict annual charges based on attributes like age, BMI, smoker status, number of children, and region. The project includes:

- Data cleaning and wrangling
- Exploratory Data Analysis (EDA)
- Linear Regression modeling
- Ridge Regression with Polynomial Features

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
